CoopTrack · Guía simple

Cómo usar CoopTrack en 1 minuto.

🧭 Qué es CoopTrack

CoopTrack organiza trabajo usando cartas.

  • Carta → acción
  • Acción → jugada
  • Jugada → línea

🃏 Las cartas

  • J → crea algo
  • Q → suma personas
  • K → delega
  • A → valida

♥ ♠ ♦ ♣ Palos

  • ♥ texto
  • ♠ tareas
  • ♦ dinero
  • ♣ bienes

🔴🔵 Tipo de mazo

🔴 Rojo: simple, sin impacto real

🔵 Azul: real, con dinero y roles

🧾 Ejemplo

  1. J♠ → creo evento
  2. Q♠ → invito gente
  3. J♦ → agrego costo
  4. Q♦ → alguien paga

📌 Cómo leer una línea

Es como una conversación:

  • qué pasó
  • qué dice
  • quién participa
  • respuestas
